## 🚀 How to Use

### Initial Setup
1. Download file from link above
2. Open in Microsoft Excel
3. **Enable Macros** when prompted for VBA functionality

### Creating a Check-In Slip
1. Go to **"Form"** sheet
2. System auto-generates unique serial number
3. Enter guest details:
   - Guest name
   - Organization (select from approved list)
   - Contact details & email
   - Aadhar/Passport number
4. Enter check-in date & time
5. Save the form

### Printing Professional Slip
1. Ensure all form fields are complete
2. Press **Ctrl+P** to print
3. Professional slip prints with all details
4. Guest & staff sign the slip
5. File physical hardcopy

### Recording in Visitor Register
1. Go to **"Data"** sheet
2. Enter guest information:
   - Check-in date
   - Guest name
   - Contact details
   - Organization (auto-validated)
   - Approval status
3. Create hyperlink to hardcopy
4. Register updated automatically

### Managing Organizations
1. Go to **"Validation"** sheet
2. View pre-approved organizations
3. Add new organizations as needed
4. Update approval status
5. Available in Form dropdown instantly
